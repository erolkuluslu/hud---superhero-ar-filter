PENÇE OYUNU - WINDOWS OTOMATIK BAŞLATMA
=======================================

KURULUM
-------
1. Win + R bas → shell:startup yaz → Enter

2. Açılan klasörde:
   - Sağ tıkla → Yeni → Kısayol
   - Konum: wscript.exe "TAM_YOL\baslat-sessiz.vbs"
   - Örnek: wscript.exe "C:\Users\Erol\hud-+-superhero-ar-filter\windows-otomasyon\baslat-sessiz.vbs"
   - İsim: Pençe Oyunu
   - Son

3. Bilgisayarı yeniden başlat


NOTLAR
------
- Oyun tam ekran açılır (kiosk modu)
- Sunucu hazır olana kadar bekler, sonra açılır
- Çıkış: Alt+F4 veya Ctrl+W


DEVRE DIŞI BIRAKMA
------------------
Win + R → shell:startup → Kısayolu sil
